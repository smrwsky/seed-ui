# Checkbox

Input of type checkbox

## When To Use

- When the user needs to select one or more options from a list of options.

- When the user can choose between two mutually exclusive options.

## Guidelines

- Use Text component for providing a label that accurately describes
the purpose of the input. Text component should be associated
with the Checkbox component using the "htmlFor" and "id" properties.

- Use ErrorMessage component for providing error feedback to indicate
that the input value is invalid.
